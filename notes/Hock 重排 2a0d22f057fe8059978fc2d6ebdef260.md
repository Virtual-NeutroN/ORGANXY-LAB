# Hock 重排

重要程度: ⭐

## 简介

![IMG_2033.jpeg](Hock%20%E9%87%8D%E6%8E%92/IMG_2033.jpeg)

有过氧或过氧氢官能团的有机化合物，在酸性条件下**烷基或芳基能迁移到缺电子性氧原子上**，此反应称为**Hock重排**。

## 机理

![JPEG图像.jpeg](Hock%20%E9%87%8D%E6%8E%92/JPEG%E5%9B%BE%E5%83%8F.jpeg)

::: tip 特点

第一步：苯经傅-克烷基化得到异丙苯；

第二步：异丙苯在催化剂作用下与氧气反应得到异丙苯过氧化氢；

第三步：异丙苯过氧化氢在酸性条件下发生Hock重排，得到丙酮和苯酚。

:::

::: warning 注意事项

该反应是一个1，2迁移重排，没错！

:::
