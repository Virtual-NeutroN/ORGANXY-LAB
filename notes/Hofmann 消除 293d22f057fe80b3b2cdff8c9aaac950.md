# Hofmann 消除

重要程度: ⭐⭐⭐

## 简介

Hofmann消除反应是指烷基三甲铵盐（季铵盐）在碱性和加热条件下消去一分子胺形成烯烃的反应，也称Hofmann降解，为从胺合成烯烃的方法之一。

![hevc.png](Hofmann%20%E6%B6%88%E9%99%A4/hevc.png)

不对称胺反应时，反应由动力学控制，较少烷基取代的β-碳上的氢由于酸性较强，位阻较小，因此优先被消除，产物主要是不稳定的取代较少的烯烃。

这个规则与扎伊采夫规则相反，称为霍夫曼规则（Hofmann规则）。

## 机理

Hofmann消除反应一般经历E2机理，但在某些情况下，反应也可能经历E1cb。反应的立体化学为反侧消除。

![hevc.jpeg](Hofmann%20%E6%B6%88%E9%99%A4/hevc.jpeg)

::: tip 特点

β-氨基酮形成季铵盐后可消除形成α,β-不饱和酮，因而，Hofmann消除常用于α,β-不饱和酮的合成。

:::

::: warning 注意事项

一定要确定，是否为

反式，共平面（比如说椅式构象下的立体化学（a键允许，e是禁止的））

:::
