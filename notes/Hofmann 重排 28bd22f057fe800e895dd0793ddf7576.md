# Hofmann 重排

中文: 霍夫曼重排
重要程度: ⭐⭐
别称: Hofmann降键

## 简介

![hevc.png](Hofmann%20%E9%87%8D%E6%8E%92/hevc.png)

霍夫曼重排反应，是一级酰胺重排变为伯胺并减少一个碳原子的有机反应。

这个反应重排出来的n=c=o水解得到的不稳定化合物，于是便成为co2离去，因此看起来不是重排

## 机理

![hevc.png](Hofmann%20%E9%87%8D%E6%8E%92/hevc%201.png)

::: tip 特点

酰胺α-碳如果有手性，重排后构型保持。α,β-不饱和酰胺或α-羟基酰胺进行Hofmann重排会生成醛酮。

可类比贝克曼重排（比较类似）

[Beckmann重排](Beckmann%E9%87%8D%E6%8E%92%2027ed22f057fe801a9968fea5b52592d2.md) 

:::

::: warning 注意事项

霍夫曼重排反应最早是以溴素为氧化剂，但是溴素的反应性很强，反应过程剧烈，副反应较多。后来的研究发现次氯酸钠、NBS、三氯异氰尿酸(TCCA)、三溴异氰尿酸(TBCA)、二氯二甲基海因(DCDMH)、四乙酸铅、高价碘试剂等也能作为氧化剂

:::
