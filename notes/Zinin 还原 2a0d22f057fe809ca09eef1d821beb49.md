# Zinin 还原

中文: 齐宁还原
重要程度: ⭐⭐

## 简介

**Zinin还原反应**是一种使用硫化物（‌如Na2S, NaHS, Na2S2(过硫化钠), Na2S2O4(连二亚硫酸钠，‌俗称保险粉)）‌作为还原剂的化学还原方法，‌主要用于芳香硝基化合物的还原。‌这种还原反应可以通过控制硫化物的量和反应温度来实现硝基化合物的选择性还原或全部还原，可用于多硝基化合物选择性还原或全部还原，只需要控制硫化物（一般过量5~10%）的量和反应温度即可。

## 机理

Ar-NO2 + 3Na2S + 4H2O ➡️ Ar-NH2 + 3S⬇️ + 6NaOH

S + Na2S ➡️ Na2S2

6S + 6NaOH ➡️ Na2S2O3 + 2Na2S2 + 3H2O

从上述反应方程式可以看出，硝基在EtOH/H2O体系中还原是一个循环自催化的过程，实质上起到还原作用的是S22-，还原过程中会不断生成S22-，直至硝基化合物被彻底还原成伯胺。

::: tip 特点

将硝基（NO2）还原为氨基（NH2）是一种十分经典的化学转化，早在一百多年前便开始有人研究。

最经典的两种还原硝基苯的方法：

一种是使用化学计量的Na2S、(NH4)2S、NH4HS或其他多硫化物作为还原剂，质子型溶剂（如H2O）作为H原子来源来实现（Zinin还原）

另一种是利用金属催化下的bechamp还原
