# [2,3]-Wittig 重排

## 简介

![hevc.jpeg](%5B2,3%5D-Wittig%20%E9%87%8D%E6%8E%92/hevc.jpeg)

强碱作用下，烯丙醚去质子后，经过五元环过渡态，[2，3]-σ重排形成高烯丙醇，为[2，3]-Wittig重排反应。也被称为Still-Wittig重排。反应机理同Sommelet-Hauser重排类似。

## 机理

![hevc.jpeg](%5B2,3%5D-Wittig%20%E9%87%8D%E6%8E%92/hevc%201.jpeg)

::: tip 特点

[2，3]-σ重排形成的烯烃产物，具有高度立体选择性，一般来说Z-式双键对应syn产物而E-式双键对应anti产物

:::

::: warning 注意事项

![hevc.jpeg](%5B2,3%5D-Wittig%20%E9%87%8D%E6%8E%92/hevc%202.jpeg)

烯丙醚进行[2，3]-σ重排时，可以会有[1，2]-σ重排的副产物（见另一个章节），两者是竞争反应。

:::
